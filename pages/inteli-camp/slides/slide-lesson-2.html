Anatomia de um bom prompt

Da spec para a instrução — cada campo vira uma parte do prompt

Prompts · Conceito

🔧 As 5 partes de um prompt eficaz

  1. Papel (Role): quem é a IA neste contexto — especialista, assistente, revisor
  2. Contexto: o que a IA precisa saber sobre o problema e o parceiro
  3. Tarefa: o que exatamente ela deve fazer — verbo claro, sem ambiguidade
  4. Parâmetros: as variáveis que mudam a cada uso (entrada do usuário, configurações)
  5. Restrições: o que ela NÃO pode fazer — derivado direto da spec

📋 Exemplo — spec → prompt

Spec diz:
Ator: analista de marketing
Entrada: briefing do produto
Saída: 3 variações de copy
Restrição: sem palavras proibidas, max 280 caracteres
Prompt gerado:
Você é um redator especialista em [setor do parceiro].
Com base no briefing: [briefing], crie
3 variações de copy para redes sociais.
Regras: máx. 280 caracteres, sem [palavras proibidas].

Como iterar o prompt até funcionar

O prompt raramente fica bom na primeira versão — e tudo bem

Prompts · Método

🔁 O ciclo de iteração

  • Escreva a primeira versão do prompt com as 5 partes
  • Teste com pelo menos 3 entradas diferentes
  • Compare o output com o critério de aceite da spec
  • Identifique qual parte do prompt causou o desvio
  • Ajuste apenas essa parte e teste novamente

⚠️ Erros mais comuns

  • Prompt vago: "escreva algo sobre o produto" — a IA inventa
  • Contexto ausente: a IA não sabe quem é o parceiro nem o público
  • Sem restrições: a IA ignora limites de tom e formato
  • Mudar várias partes de uma vez: impossível saber o que funcionou

🎯 Como usar os parâmetros

  • Identifique o que muda a cada uso: é o parâmetro
  • Marque com colchetes no prompt: [briefing], [tom], [produto]
  • No Lovable, cada parâmetro vira um campo de formulário
  • Teste o mesmo prompt com parâmetros diferentes para validar

🏁 Entregável do dia

Critério de conclusão
O prompt está pronto quando o output de 3 entradas diferentes atende ao critério de aceite definido na spec do Dia 1. Se não atender, o ajuste é no prompt, não no critério.
