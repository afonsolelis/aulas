Objetivo da Aula

Compreender os fundamentos das principais estruturas de dados, identificar suas complexidades algorítmicas e implementar exemplos práticos em Python.

1) O que são Estruturas de Dados?

Estruturas de dados são formas organizadas de armazenar e acessar informações na memória. A escolha correta impacta diretamente a eficiência do sistema.

  • Lineares: Arrays, Linked Lists, Stacks, Queues
  • Não-lineares: Trees, Graphs, Heaps
  • Hash-based: Hash Tables, Dictionaries

2) Estruturas Lineares

Array

Coleção de elementos em posições contíguas de memória. Acesso por índice em O(1), inserção/remoção no meio em O(n).

nums = [10, 20, 30, 40]
print(nums[2])   # O(1) → 30
nums.insert(1, 15)  # O(n) → [10, 15, 20, 30, 40]
Stack (LIFO)

Pilha — o último a entrar é o primeiro a sair. Usado em chamadas de função, undo/redo, parsing de expressões.

stack = []
stack.append("a")  # push
stack.append("b")
top = stack.pop()  # "b" — O(1)
Queue (FIFO)

Fila — o primeiro a entrar é o primeiro a sair. Usado em agendamento de tarefas, BFS, filas de mensagens.

from collections import deque
q = deque()
q.append("task1")
q.append("task2")
first = q.popleft()  # "task1" — O(1)

3) Árvores e Grafos

Binary Search Tree (BST): estrutura onde nó esquerdo < raiz < nó direito. Busca em O(log n) na média.

class Node:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

def insert(root, val):
    if root is None:
        return Node(val)
    if val < root.val:
        root.left = insert(root.left, val)
    else:
        root.right = insert(root.right, val)
    return root

Grafos: conjunto de vértices e arestas. Usados para redes sociais, rotas, dependências de pacotes.
