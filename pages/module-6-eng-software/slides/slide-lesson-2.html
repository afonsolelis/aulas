Notebook · Célula 4 · predict(user, item)
Média ponderada dos ratings que o usuário deu a itens parecidos com o alvo
In [4]:
def predict(user, item, k=3): u = ratings.index.get_loc(user) i = ratings.columns.get_loc(item) vistos = np.where(~np.isnan(R[u]))[0] sims = S.values[i, vistos] top = np.argsort(-sims)[:k] j = vistos[top] num = np.sum(S.values[i, j] * R[u, j]) den = np.sum(np.abs(S.values[i, j])) return num / den if den > 0 else np.nan predict('Ana', 'Bossa'), predict('Ana', 'Pop')
▶ Running
Out[4]:
(3.42, 4.18)

📐 Leitura: a função usou os 3 itens mais parecidos com o alvo entre os que Ana já viu, ponderou os ratings dela por essas similaridades — Bossa ficou em 3.42, Pop em 4.18. Pop deve ser recomendado primeiro.

Notebook · Célula 5 · top_k_recommend('Ana')
Filtra os já vistos, aplica predict em cada candidato, ranqueia top-N
In [5]:
def top_k_recommend(user, k=10, n=3): vistos = ratings.loc[user].dropna().index candidatos = ratings.columns.difference(vistos) scores = [(predict(user, item, k), item) for item in candidatos] return sorted(scores, reverse=True)[:n] top_k_recommend('Ana', n=3)
▶ Running
Out[5]:
[(4.18, 'Pop'), (3.42, 'Bossa')]
🎯 Recomendação para Ana: assistir Pop primeiro, depois Bossa
Notebook · Célula 6 · Avaliação RMSE / MAE · 80/20
Mascarar 20% dos ratings observados como teste · medir erro
In [6]:
from sklearn.model_selection import train_test_split from sklearn.metrics import mean_squared_error, mean_absolute_error # mascarar 20% dos ratings observados como teste mask = ~np.isnan(R) idx = np.argwhere(mask) train_idx, test_idx = train_test_split(idx, test_size=0.2, random_state=42) y_true, y_pred = [], [] for u, i in test_idx: user = ratings.index[u]; item = ratings.columns[i] y_true.append(R[u, i]) y_pred.append(predict(user, item)) rmse = np.sqrt(mean_squared_error(y_true, y_pred)) mae = mean_absolute_error(y_true, y_pred) print(f"RMSE: {rmse:.2f}"); print(f"MAE: {mae:.2f}")
▶ Running
Out[6]:
RMSE: 0.94 MAE: 0.71

📏 Leitura: MAE de ~0.7 em escala 1–5 = errar menos de uma estrela em média. Para um motor item-based simples, sem fine-tuning, é resultado decente. É esse o tipo de número que vocês vão reportar na atividade ponderada.
