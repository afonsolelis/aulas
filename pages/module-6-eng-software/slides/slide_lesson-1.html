📖 O que é?

O SWEBOK é o corpo de conhecimento publicado pelo IEEE que organiza 15 áreas da engenharia de software, servindo como referência para profissionais e educadores.

🎯 Por que importa?

Fornece vocabulário comum para a área, guia melhores práticas reconhecidas internacionalmente e fundamenta certificações profissionais como o CSEP.

SEBOK — Systems Engineering Body of Knowledge
Pensando em sistemas completos, não apenas software

📖 O que é SEBOK?

Guia de conhecimento publicado pelo INCOSE/BKCASE que cobre engenharia de sistemas como disciplina. Foca em como projetar, integrar e gerenciar sistemas complexos onde software é apenas uma parte do todo.

🔄 SEBOK vs SWEBOK

SWEBOK foca em software como produto. SEBOK foca no sistema como um todo (hardware + software + pessoas + processos). No nosso projeto, precisamos dos dois: SWEBOK para construir o app, SEBOK para integrar tudo.

React Native — Mobile com JavaScript
Um framework, dois sistemas operacionais

📱 O que é?

Framework open-source criado pelo Meta (Facebook) em 2015. Permite construir apps nativos para iOS e Android usando JavaScript e React. Diferente de webviews — compila para componentes nativos reais.

🚀 Por que React Native?

  • Uma única codebase para iOS e Android
  • Hot reload para desenvolvimento rápido
  • Ecossistema imenso de pacotes npm
  • Usado por Instagram, Discord, Shopify, Uber Eats

Filtragem Colaborativa por Usuário
O coração do sistema de recomendação

💡 O conceito

"Usuários parecidos gostam de coisas parecidas."

Se Ana e Bruno deram nota alta para os mesmos filmes, e Ana gostou de um filme que Bruno não viu, recomendamos esse filme para Bruno.

🎯 No Projeto 6

Vamos implementar essa lógica em um backend (API RESTful) e exibir as recomendações no app React Native.

Pipeline completo: coleta → similaridade → predição → exibição.

1
📥 Coleta de avaliações dos usuários
Ratings explícitos (1-5 estrelas) ou implícitos (cliques, tempo)
2
📊 Cálculo de similaridade (ex: cosseno)
Mede o quão parecidos são dois usuários com base nas avaliações
3
🔮 Predição de notas para itens não avaliados
Média ponderada das notas dos vizinhos mais similares
4
📱 Exibição no app mobile via API
Top-N recomendações renderizadas no React Native
