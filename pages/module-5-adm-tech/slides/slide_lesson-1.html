Medidas de Tendência Central

📊 MÉDIA

Soma ÷ quantidade. Sensível a outliers.

=MÉDIA(intervalo)

📏 MEDIANA

Valor central ordenado. Robusta a outliers.

=MED(intervalo)

🎯 MODA

Valor mais frequente. Dados categóricos.

=MODO(intervalo)
Quando usar cada uma?
Média: dados simétricos | Mediana: com outliers (salários, imóveis) | Moda: categóricos

Variações da Função MÉDIA

FunçãoSintaxeUso
MÉDIA=MÉDIA(A1:A100)Média aritmética simples
MÉDIASE=MÉDIASE(intervalo;critério)Média condicional (1 critério)
MÉDIASES=MÉDIASES(interv;c1;i1;...)Média condicional (múltiplos)
MÉDIA.GEOMÉTRICA=MÉDIA.GEOMÉTRICA(A1:A10)Taxas de crescimento
MÉDIA.HARMÔNICA=MÉDIA.HARMÔNICA(A1:A10)Velocidades e taxas
// Média de vendas do Produto A na região Sul
=MÉDIASES(D2:D100; B2:B100; "Produto A"; C2:C100; "Sul")

Medidas de Dispersão

Por que a média não é suficiente?

Duas turmas têm média 7.0. São iguais?
Turma A: 6, 7, 7, 7, 8 → Média = 7 (homogênea)
Turma B: 2, 5, 7, 9, 12 → Média = 7 (muito variável!)

📊 VARIÂNCIA

Média dos quadrados dos desvios.

=VAR(intervalo)

📏 DESVIO PADRÃO

Raiz da variância. Mesma unidade dos dados.

=DESVPAD(intervalo)

📐 COEF. VARIAÇÃO

DP ÷ Média. Compara escalas diferentes.

=DESVPAD(A:A)/MÉDIA(A:A)

Quartis e Percentis

=QUARTIL(intervalo; quarto) — Retorna Q0 a Q4
=PERCENTIL(intervalo; k) — Retorna o percentil k (0 a 1)
QuartilPercentilSignificadoFórmula
Q1P2525% dos dados abaixo=QUARTIL(A:A;1)
Q2 (Mediana)P5050% dos dados abaixo=MED(A:A)
Q3P7575% dos dados abaixo=QUARTIL(A:A;3)
// Amplitude Interquartil (IQR)
=QUARTIL(A:A;3) - QUARTIL(A:A;1)

// Detectar outliers (Método Tukey)
=SE(OU(A2 < Q1-1.5*IQR; A2 > Q3+1.5*IQR); "Outlier"; "Normal")

Lei dos Grandes Números

📜 Definição

À medida que o número de tentativas aumenta, a média amostral converge para o valor esperado (média populacional).

🎯 Lançamento de Moeda

  • 10 lançamentos: pode dar 70% cara
  • 100 lançamentos: ~55% cara
  • 1.000 lançamentos: ~51% cara
  • 10.000 lançamentos: ~50.1% cara

💼 Aplicações em Negócios

  • Seguradoras: Precificação de apólices
  • Cassinos: Margem garantida no longo prazo
  • Pesquisas: Tamanho mínimo de amostra
  • Trading: Estratégias de longo prazo
// Simulando LGN no Excel
=ALEATÓRIOENTRE(0;1) // Col B: lançamento de moeda
=MÉDIA($B$1:B1) // Col C: média acumulada — converge para 0.5!

Previsão e Forecast

📈 TENDÊNCIA

Regressão linear múltipla. Ideal para múltiplas variáveis independentes.

=TENDÊNCIA(y_conhecidos; x_conhecidos; novos_x)

🔮 PREVISÃO.LINEAR

Projeta um único ponto futuro com base na regressão linear.

=PREVISÃO.LINEAR(x; y_conhecidos; x_conhecidos)

📊 CRESCIMENTO

Regressão exponencial. Para dados com crescimento não-linear.

=CRESCIMENTO(y_conhecidos; x_conhecidos; novos_x)
Quando usar: Linear para tendências constantes. Exponencial para crescimento acelerado (receita de startups, dados populacionais).

Valuation: DCF, VPL e TIR

💰 VPL — Valor Presente Líquido

Traz fluxos futuros a valor de hoje usando uma taxa de desconto (TMA).

=VPL(taxa; fluxos) + investimento_inicial

VPL > 0: projeto viável | VPL < 0: destruição de valor

📈 TIR — Taxa Interna de Retorno

A taxa de desconto que zera o VPL. Compara com o custo de capital.

=TIR(fluxos)

TIR > TMA: viável | TIR < TMA: inviável
