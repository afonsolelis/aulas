Passo 1: Excel → CSV

Checklist de exportação

  • Salvar a aba correta: Arquivo → Salvar como → CSV UTF-8.
  • Uma linha por período, colunas com nomes curtos e claros.
  • Sem células mescladas, sem fórmulas quebradas, sem totais no meio da tabela.
  • Separador decimal consistente (preferência: ponto para Python).

Colunas mínimas sugeridas:

periodo,investimento_inicial,entrada,saida,fluxo_caixa 2025-01,-150000,0,0,-150000 2025-02,0,28000,12000,16000 2025-03,0,32000,13000,19000 2025-04,0,35000,14000,21000 2025-05,0,36000,15000,21000 2025-06,0,38000,15000,23000
Observação: para VPL/TIR basta uma coluna de fluxo de caixa.

Passo 2: Organizar com Pandas

Leitura e limpeza básica

import pandas as pd # Upload do CSV no Colab df = pd.read_csv("fluxo_caixa_projeto.csv") # Tipos e consistência df["periodo"] = pd.to_datetime(df["periodo"]) df["fluxo_caixa"] = pd.to_numeric(df["fluxo_caixa"], errors="coerce") # Remover linhas inválidas df = df.dropna(subset=["fluxo_caixa"]).sort_values("periodo") display(df)
Depois da limpeza, use sempre df.info() para validar tipos antes dos cálculos.

Passo 3: VPL, TIR e Payback

!pip install numpy-financial -q import numpy as np import numpy_financial as npf fluxos = df["fluxo_caixa"].to_list() tma = 0.012 # 1,2% ao mês # VPL vpl = npf.npv(tma, fluxos) # TIR tir = npf.irr(fluxos) # Payback simples (em períodos) acumulado = np.cumsum(fluxos) payback_idx = np.argmax(acumulado >= 0) payback = int(payback_idx) if acumulado[payback_idx] >= 0 else None print(f"VPL: R$ {vpl:,.2f}") print(f"TIR: {tir:.2%}") print(f"Payback: {payback} períodos")
