Problema: Dados Esparsos

Registros financeiros frequentemente chegam sem periodicidade regular. Estrutura: Data, Projeto, Tipo, Valor

Solução: Tabela Dinâmica para agrupar por Projeto e somar por ano.

Tabela Dinâmica no Google Sheets

Consolidar por ano para preparar o VPL:

  1. Selecione a aba Fluxo_Detalhado completa.
  2. Menu: Inserir → Tabela dinâmica (nova aba).
  3. Linhas: Data (agrupar por ano).
  4. Valores: Soma de Valor.
  5. Filtro (opcional): Projeto = A.
Resultado: fluxo líquido anual consolidado.
Exemplo: 2024 → -109.000 | 2025 → 40.000 | 2026 → 67.000 | 2027 → 91.000

Cálculo do VPL com Dados Consolidados

// Estrutura na aba VPL
B1: taxa de desconto (ex.: 0,12)
C2: fluxo inicial — ano 0, negativo (-120000)
C3:C6: fluxos anuais futuros consolidados
B3: resultado do VPL
Fórmula:
=VPL(B1; C3:C6) + C2
Checagem didática: confirmar sinal do investimento inicial (negativo) e periodicidade anual. Se VPL > 0, o projeto é viável à taxa escolhida.

Ponto de Fisher

O cruzamento onde dois projetos têm o mesmo VPL. Abaixo dessa taxa, um projeto é preferível; acima, o outro.

  • Taxa < Fisher: Prefira o projeto de maior investimento inicial.
  • Taxa > Fisher: Prefira o projeto de retorno mais rápido.
  • Ferramenta: Tabela de sensibilidade (taxa de 0% a 16%).
A metodologia Stage-Gate usa o perfil do VPL para decisões Go/Kill nos portões de projeto.

Macro no Google Sheets: Automação do VPL

// Apps Script — calcularVPLProjeto
function calcularVPLProjeto() {
var sheet = SpreadsheetApp.getActiveSheet();
var taxa = sheet.getRange("B1").getValue();
var fluxoInicial = sheet.getRange("C2").getValue();
var fluxos = sheet.getRange("C3:C6").getValues().flat();

var vpl = fluxoInicial;
for (var i = 0; i < fluxos.length; i++) {
vpl += fluxos[i] / Math.pow(1 + taxa, i + 1);
}

sheet.getRange("B3").setValue(vpl);
sheet.getRange("B3").setNumberFormat("R$ #,##0.00");
Logger.log("VPL calculado: " + vpl);
}
Critério de sucesso: macro funcional — recálculo do VPL sem editar fórmula manualmente. Entregue os 3 exercícios e a macro na planilha.
