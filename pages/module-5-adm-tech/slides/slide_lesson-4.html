VPL: Implementação Manual

def calcular_vpl(fluxos_caixa: list[float], taxa_desconto: float) -> float: vpl = 0 for t, fluxo in enumerate(fluxos_caixa): vpl += fluxo / ((1 + taxa_desconto) ** t) return vpl fluxos = [-500000, 130000, 150000, 175000, 200000, 215000] taxa = 0.14 vpl = calcular_vpl(fluxos, taxa) print(f"VPL: R$ {vpl:,.2f}")
Fórmula: VPL = Σ Ft / (1+r)^t para t = 0, 1, ..., n
VPL > 0 = projeto viável | VPL < 0 = destrói valor

TIR via Newton-Raphson

A TIR é a taxa r que faz VPL(r) = 0 — uma raiz da função VPL. O método de Newton-Raphson converge em geral em menos de 10 iterações.

Fórmula: taxa_nova = taxa − VPL(taxa) / VPL'(taxa)
Repete até |taxa_nova − taxa| < tolerância
def calcular_tir(fluxos_caixa: list[float], chute=0.1, tol=1e-6, max_iter=100) -> float: taxa = chute for _ in range(max_iter): vpl = sum(f / ((1 + taxa) ** t) for t, f in enumerate(fluxos_caixa)) derivada = sum(-t * f / ((1 + taxa) ** (t+1)) for t, f in enumerate(fluxos_caixa) if t > 0) if abs(derivada) < 1e-10: break nova = taxa - vpl / derivada if abs(nova - taxa) < tol: return nova taxa = nova return taxa tir = calcular_tir(fluxos) print(f"TIR: {tir*100:.2f}% a.a.")

Payback: Simples e Descontado

Payback Simples

Período em que o fluxo acumulado iguala o investimento inicial (nominal).

Payback Descontado

Mesma lógica, usando fluxos trazidos a valor presente. Sempre maior que o simples.

def calcular_payback_simples(fluxos_caixa: list[float]) -> float: inv = -fluxos_caixa[0] acum = 0 for t in range(1, len(fluxos_caixa)): acum += fluxos_caixa[t] if acum >= inv: frac = (inv - (acum - fluxos_caixa[t])) / fluxos_caixa[t] return (t - 1) + frac return float('inf') def calcular_payback_descontado(fluxos_caixa: list[float], taxa: float) -> float: inv = -fluxos_caixa[0] acum = 0 for t in range(1, len(fluxos_caixa)): fd = fluxos_caixa[t] / ((1 + taxa) ** t) acum += fd if acum >= inv: return (t - 1) + (inv - (acum - fd)) / fd return float('inf')

Integrando Todas as Funções

fluxos_completos = [-500000, 130000, 150000, 175000, 200000, 215000] taxa_minima = 0.14 vpl = calcular_vpl(fluxos_completos, taxa_minima) tir = calcular_tir(fluxos_completos) pb_s = calcular_payback_simples(fluxos_completos) pb_d = calcular_payback_descontado(fluxos_completos, taxa_minima) print(f"VPL: R$ {vpl:,.2f}") print(f"TIR: {tir*100:.2f}% a.a.") print(f"Payback Simples: {pb_s:.2f} anos") print(f"Payback Descontado: {pb_d:.2f} anos") if vpl > 0 and tir > taxa_minima: print("Projeto VIAVEL") else: print("Projeto INVIAVEL")

Visualização com Plotly

import plotly.graph_objects as go def plotar_sensibilidade_vpl(fluxos, taxa_min=0, taxa_max=0.30): taxas = [i/100 for i in range(int(taxa_min*100), int(taxa_max*100)+1)] vpls = [calcular_vpl(fluxos, t) for t in taxas] fig = go.Figure() fig.add_trace(go.Scatter(x=taxas, y=vpls, mode='lines', line=dict(color='#2e2640', width=3))) fig.add_hline(y=0, line_dash="dash", line_color="red") fig.update_layout(title='Sensibilidade do VPL a Taxa de Desconto', xaxis_title='Taxa de Desconto', yaxis_title='VPL (R$)', hovermode='x unified') return fig fig = plotar_sensibilidade_vpl(fluxos_completos) fig.show()
O cruzamento da curva com y=0 é a TIR — onde o projeto deixa de ser viável.

App Completo com Gradio
