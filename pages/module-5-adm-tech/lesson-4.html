🧩 Bloco 1: Preparação dos Dados
XLSX → CSV

Por que começar pela estrutura da planilha?

Problemas comuns

  • Células mescladas no cabeçalho
  • Duas tabelas na mesma aba
  • Totais no meio dos dados
  • Colunas com formatos mistos

Impacto no código

  • Leitura quebrada no `read_csv`
  • Tipos errados (`object` em números)
  • Resultados incorretos em cálculos
  • Pipeline difícil de manter

Pipeline mínimo de qualidade

# padroniza nomes das colunas
df.columns = [c.strip().lower() for c in df.columns]

# converte colunas numéricas
num_cols = ["receita", "custos", "investimento", "taxa_desconto"]
for col in num_cols:
  df[col] = pd.to_numeric(df[col], errors="coerce")

# remove linhas inválidas
df = df.dropna(subset=["ano", "receita", "custos"])

Ponto crítico: use `errors="coerce"` para detectar lixo textual nas colunas numéricas.

💰 Bloco 3: Lógica de Valuation
Python

Modelo simplificado de VPL

Vamos calcular um VPL com base em fluxo de caixa livre anual:

FCL_t = receita_t - custos_t
VPL = investimento_inicial + Σ(FCL_t / (1 + taxa)^t)

Assumimos `investimento` negativo no ano 0 e fluxos positivos/negativos nos anos seguintes.

🧠 Função Python Reutilizável
Implementação

Função base para usar no notebook e no app

import pandas as pd

def calcular_vpl(df: pd.DataFrame, taxa_desconto: float) -> tuple[float, pd.DataFrame]:
  base = df.copy()
  base = base.sort_values("ano")
  base["fcl"] = base["receita"] - base["custos"]
  base["fator_desconto"] = (1 + taxa_desconto) ** base["ano"]
  base["vp_fcl"] = base["fcl"] / base["fator_desconto"]

  investimento_inicial = base.loc[base["ano"] == 0, "investimento"].sum()
  vpl = investimento_inicial + base["vp_fcl"].sum()
  return float(vpl), base

Por que Gradio neste contexto?

⚙️ Estrutura do App
Upload + Cálculo

Componentes mínimos da interface

  • Input: upload de CSV
  • Input: taxa de desconto (`number` ou `slider`)
  • Output: valor do VPL
  • Output: tabela de projeções processadas
  • Output opcional: mensagem de erro tratada

Contrato da função do app

A função chamada pelo Gradio deve receber arquivo + taxa e retornar texto + DataFrame.

import gradio as gr
import pandas as pd

def processar_csv(arquivo, taxa):
  try:
    df = pd.read_csv(arquivo.name)
    vpl, proj = calcular_vpl(df, float(taxa))
    msg = f"VPL calculado: R$ {vpl:,.2f}"
    return msg, proj
  except Exception as e:
    return f"Erro no processamento: {e}", pd.DataFrame()

demo = gr.Interface(
  fn=processar_csv,
  inputs=[gr.File(label="Arquivo CSV"), gr.Number(value=0.14, label="Taxa de desconto")],
  outputs=[gr.Textbox(label="Resultado"), gr.Dataframe(label="Projeções")],
  title="Valuation Simplificado com CSV"
)
demo.launch(share=True)
