üîÑ Resampling - Mudan√ßa de Frequ√™ncia
Bloco 2

Agregando dados em diferentes frequ√™ncias

Resampling converte dados de uma frequ√™ncia para outra (ex: di√°rio ‚Üí mensal).

Downsampling (aumentar intervalo)

# De di√°rio para mensal

# De di√°rio para semanal

  'Open': 'first', # Primeira abertura da semana
  'High': 'max', # M√°xima da semana
  'Low': 'min', # M√≠nima da semana
  'Close': 'last', # √öltimo fechamento
  'Volume': 'sum' # Volume total
})

Upsampling (diminuir intervalo)

# De mensal para di√°rio (precisa preencher gaps)
df_daily = df_monthly.resample('D').ffill() # Forward fill
df_daily_interp = df_monthly.resample('D').interpolate() # Interpola√ß√£o

C√≥digos de frequ√™ncia

C√≥digoDescri√ß√£o
'D'Di√°rio
'W'Semanal
'M'Mensal (√∫ltimo dia)
'MS'Mensal (primeiro dia)
'Q'Trimestral
'Y'Anual
ü™ü Rolling Windows - M√©dias M√≥veis
Bloco 2

C√°lculos em janelas deslizantes

Rolling windows s√£o essenciais para m√©dias m√≥veis, volatilidade m√≥vel, e outros indicadores t√©cnicos.

M√©dias m√≥veis simples

# M√©dia m√≥vel exponencial (EMA)

Volatilidade m√≥vel (desvio padr√£o)

# Volatilidade m√≥vel de 20 dias (anualizada)

Fun√ß√µes agregadas em rolling

# M√°xima e m√≠nima dos √∫ltimos 52 dias (highs/lows de 52 semanas)

df['low_52w'] = df['Low'].rolling(window=252).min()

# Fun√ß√µes customizadas
df['range_20d'] = df['Close'].rolling(window=20).apply(
  lambda x: x.max() - x.min()
