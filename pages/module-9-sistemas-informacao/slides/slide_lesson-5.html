As 3 Categorias de Padrões GoF

🏭 Criacionais

Como objetos são criados. Desacoplam o cliente da classe concreta.

  • Singleton
  • Factory Method
  • Abstract Factory
  • Builder, Prototype

🏗️ Estruturais

Como classes e objetos são compostos em estruturas maiores.

  • Decorator
  • Adapter
  • Composite
  • Facade, Proxy

🎭 Comportamentais

Como objetos colaboram e distribuem responsabilidades.

  • Observer
  • Strategy
  • Command
  • Iterator, Template Method

Singleton (Criacional)

🎯 Problema

Garantir que uma classe tenha apenas uma instância e fornecer um ponto global de acesso a ela.

Casos de uso: Logger, configurações, pool de conexões.

💻 Implementação C#

public class Logger {
private static Logger? _instance;
private Logger() {}
public static Logger Instance =>
_instance ??= new Logger();
public void Log(string msg) =>
Console.WriteLine($"[LOG] {msg}");
}

Observer (Comportamental)

📡 Problema

Quando um objeto muda de estado, todos os seus dependentes devem ser notificados automaticamente.

Casos de uso: Eventos de UI, notificações, EDA.

💻 Implementação C#

public interface IObservador {
void Atualizar(string evento);
}
public class Estoque : IObservavel {
private List<IObservador> _obs = new();
public void Inscrever(IObservador o) =>
_obs.Add(o);
public void Notificar(string e) =>
_obs.ForEach(o => o.Atualizar(e));
}

Strategy (Comportamental)

🎯 Problema

Definir uma família de algoritmos, encapsular cada um e torná-los intercambiáveis sem alterar o cliente.

Casos de uso: Cálculo de frete, ordenação, autenticação.

💻 Implementação C#

public interface ICalculaFrete {
decimal Calcular(Pedido p);
}
public class FreteCorreios : ICalculaFrete {
public decimal Calcular(Pedido p) => p.Peso * 2.5m;
}
public class FreteExpress : ICalculaFrete {
public decimal Calcular(Pedido p) => p.Peso * 5.0m;
}

Decorator (Estrutural)

🎨 Problema

Adicionar responsabilidades a um objeto dinamicamente, sem alterar sua classe. Alternativa flexível à herança.

Casos de uso: Logging, cache, validação, compressão.

💻 Implementação C#

public class ServicoComLog : IServico {
private readonly IServico _servico;
private readonly ILogger _log;
public ServicoComLog(IServico s, ILogger l) {
_servico = s; _log = l;
}
public void Executar() {
_log.Log("Iniciando...");
_servico.Executar();
_log.Log("Concluído.");
}
}
