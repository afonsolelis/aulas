RESTful API em .NET

📋 Princípios REST

  • Stateless: cada request é independente
  • Recursos identificados por URI
  • Verbos HTTP: GET, POST, PUT, DELETE
  • Representação em JSON/XML

💻 Controller em .NET

[ApiController]
[Route("api/[controller]")]
public class PedidosController : ControllerBase {
[HttpPost]
public async Task<IActionResult> Criar(
[FromBody] CriarPedidoDto dto) {
var id = await _service.CriarAsync(dto);
return CreatedAtAction(nameof(Obter), new { id }, null);
}
}

gRPC: Alta Performance com Protobuf

⚡ Vantagens do gRPC

  • Protobuf: serialização binária muito mais rápida
  • Contrato fortemente tipado (.proto)
  • HTTP/2: multiplexação de streams
  • Geração automática de cliente e servidor

📄 Definição .proto

syntax = "proto3";
service PedidoService {
rpc CriarPedido (CriarPedidoRequest)
returns (CriarPedidoResponse);
}
message CriarPedidoRequest {
string cliente_id = 1;
double total = 2;
}

MediatR e CQRS em .NET

📨 MediatR

Implementa o padrão Mediator. Desacopla quem envia um comando/query de quem o processa.

  • Commands: mudam estado
  • Queries: lêem estado
  • Notifications: eventos publicados
