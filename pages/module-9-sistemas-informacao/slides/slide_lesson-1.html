Agenda da Aula

Bloco 1 — Setup de Máquina

Instalação e configuração do .NET SDK 9, padronização do ambiente entre Windows, macOS e Linux.

Exploração e descoberta de dados locais e em banco usando ferramentas de IA.

Objetivos da Aula

Setup do Ambiente

  • Garantir o .NET SDK 9 instalado e funcionando
  • Padronizar o ambiente entre Windows, macOS e Linux
  • Entender diferença entre SDK e Runtime

Blazor e IA

  • Rodar um projeto Blazor com Hot Reload
  • Usar IA para discovery de dados local e em banco
  • Explorar comandos básicos do C#

Checklist de Setup

🖥️ Antes de Instalar o .NET

  • Verificar versão do sistema operacional
  • Instalar Git e configurar usuário
  • Instalar VS Code com extensões C#
  • Verificar permissões de administrador

🛠️ Instalação do .NET SDK 9

dotnet --version
dotnet new blazor -n MeuProjeto
cd MeuProjeto
dotnet watch run

Discovery de Dados com IA

O que é Discovery de Dados?

Processo de exploração e entendimento de dados disponíveis para o projeto, utilizando ferramentas de IA para acelerar a compreensão da estrutura e qualidade dos dados.

📊 Dados Locais

Explorar arquivos CSV, JSON e Excel com auxílio de IA para entender padrões e anomalias.

🗄️ Dados em Banco

Conectar ao banco do parceiro e usar IA para descobrir relacionamentos e volumes de dados.

📝 Documentação

Registrar achados do discovery no repositório como base para o desenvolvimento do projeto.
