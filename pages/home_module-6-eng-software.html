Sobre o Módulo

Desenvolvimento de uma plataforma digital para recomendação em dispositivos móveis, com arquitetura orientada a serviços, programação orientada a objetos, TDD, padrões de design e qualidade de código.

Principais Tópicos
📋 TAPI:
Acessar Termo de Abertura de Projeto Integrador
🏗️ Arquitetura:
Serviços distribuídos, arquitetura web e requisitos
🧪 Engenharia:
POO, TDD, padrões de design e qualidade de código
📱 Produto:
Recomendação para mobile, UX, acessibilidade e marketplace
