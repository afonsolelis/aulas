Matchers do Jest
As funções de asserção que verificam o comportamento esperado
toBe()
Igualdade estrita
Usa === para comparar primitivos.
expect(soma(1,2)).toBe(3)
toEqual()
Igualdade profunda (objetos)
Compara objetos e arrays recursivamente.
expect(user).toEqual({id:1,nome:'Ana'})
toBeTruthy() / toBeFalsy()
Verdadeiro / Falso
Verifica se o valor é truthy ou falsy.
expect(lista.length).toBeTruthy()
toContain()
Array ou string contém
Array ou string contém o valor esperado.
expect(nomes).toContain('Ana')
toHaveLength()
Tamanho
Verifica o tamanho de arrays ou strings.
expect(lista).toHaveLength(3)
toThrow()
Deve lançar erro
Verifica se a função lança uma exceção.
expect(() => fn(-1)).toThrow()
toBeGreaterThan() / toBeLessThan()
Comparações numéricas
Compara magnitude de números.
expect(preco).toBeGreaterThan(0)
toBeNull() / toBeUndefined()
Nulo / Indefinido
Verifica ausência de valor com precisão.
expect(resultado).toBeNull()
