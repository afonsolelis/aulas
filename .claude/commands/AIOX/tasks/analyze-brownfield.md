# Task: analyze-brownfield

Run 10-phase technical debt assessment for existing codebases.

## Usage
`/AIOX:tasks:analyze-brownfield`

## Description
Conducts brownfield discovery across system architecture, database, and UX,
producing a technical debt report and epic/stories for remediation.
Loads full task from: `.aiox-core/development/tasks/analyze-brownfield.md`

## Phases
1-3: Data Collection | 4-7: Draft & Validation | 8-10: Finalization

## Agent
@architect (Aria) orchestrates
