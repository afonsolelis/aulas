# Task: dev-develop-story

Execute story development with selectable automation modes.

## Usage
`/AIOX:tasks:dev-develop-story [story-id]`

## Description
Implements a story following the Story Development Cycle (SDC Phase 3).
Loads full task from: `.aiox-core/development/tasks/dev-develop-story.md`

## Modes
- **Interactive** (default) — checkpoints at key decisions
- **YOLO** — autonomous, decisions logged
- **Pre-Flight** — all questions upfront, then zero-ambiguity execution
