# Task: correct-course

Course correction when development drifts from requirements.

## Usage
`/AIOX:tasks:correct-course [story-id]`

## Description
Detects scope drift, misaligned implementations, or missed acceptance criteria
and produces a correction plan.
Loads full task from: `.aiox-core/development/tasks/correct-course.md`

@dev (Dex) or @qa (Quinn)
