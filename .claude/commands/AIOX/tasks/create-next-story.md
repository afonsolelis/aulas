# Task: create-next-story

Create the next story from an epic or PRD.

## Usage
`/AIOX:tasks:create-next-story [epic-id]`

## Description
Transforms epic context into a hyper-detailed story file.
Loads full task from: `.aiox-core/development/tasks/create-next-story.md`

## Output
`docs/stories/{epicNum}.{storyNum}.story.md`
