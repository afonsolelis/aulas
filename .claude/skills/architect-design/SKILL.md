# architect-design

Ativa o agente @architect (Aria) para decisões de arquitetura e design de sistema.

## Como usar

```
/architect-design [feature ou problema]
```

## Capacidades

- Decisões de arquitetura de sistema
- Seleção de tecnologias
- Avaliação de complexidade (5 dimensões, score 1-5)
- Padrões de integração
- Planejamento de implementação (implementation.yaml)
- Delegação para @data-engineer (DDL detalhado)

## Quando usar

- Nova funcionalidade com impacto arquitetural
- Avaliação de risco técnico
- Spec Pipeline Phase 2 e 6
- Brownfield Discovery Phase 1, 4, 8
