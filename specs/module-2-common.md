# Spec: Módulo 2 do Ciclo Comum

## Objetivo

Definir o padrão do módulo compartilhado entre cursos antes da separação por trilha.

## Regras visuais

1. O módulo 2 deve usar identidade cinza.
2. O card da `index.html` deve usar badge e header cinza.
3. Os botões do módulo 2 devem usar a variante comum cinza.

## Estrutura

1. O módulo deve ter home própria em `pages/home-module-2-common.html`.
2. O módulo deve ter lista de aulas com botões de `Slides` e `Material`.
3. O módulo deve ter diretórios próprios para slides e materiais.

## Conteúdo atual

1. O módulo 2 foi baseado no repositório de referência `afonsolelis/inteli_refactor`, pasta `in02`.
2. As 11 aulas devem existir no portal com links válidos.
3. Enquanto o conteúdo definitivo não for migrado, placeholders são aceitáveis.
