# Aula 7 - FRONT-END 1 - HTML, DOM E JAVASCRIPT

## Descrição da Aula
Introdução à construção da interface web com HTML, manipulação da árvore DOM e uso de JavaScript para responder a eventos e alterar a página.

---
**Visão RM-ODP:** Engineering + Technology  
**RF:** Traduzir funcionalidades em elementos visíveis e interativos  
**RNF (8 eixos):** Usabilidade + Suportabilidade  
**Artefato:** Página inicial funcional + mapeamento tela → RF

### Onde queremos chegar
- [ ] Estruturar páginas com HTML semântico
- [ ] Identificar e manipular elementos do DOM
- [ ] Responder a eventos com JavaScript
- [ ] Validar entradas simples no cliente
- [ ] Relacionar elementos da interface com RF do sistema

---
## Auto Estudo Recomendado (Tempo Estimado Total: 120 min)

1. **HTML semântico** (Estimativa: 30 min):
   - Estrutura de página, formulários, tabelas e navegação.
   **Por que estudar isso:** Cria a base da interface e melhora clareza da marcação.

2. **DOM** (Estimativa: 35 min):
   - Seleção, leitura e alteração de elementos.
   **Por que estudar isso:** Mostra como a página pode reagir às ações do usuário.

3. **Eventos em JavaScript** (Estimativa: 30 min):
   - Clique, envio de formulário e interação básica.
   **Por que estudar isso:** Permite transformar HTML estático em interface funcional.

4. **Exercícios de interação local** (Estimativa: 25 min):
   - Mostrar, esconder, validar e atualizar conteúdo.
   **Por que estudar isso:** Consolida a lógica front-end antes da integração com rede.
